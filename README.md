# Memotest

## Instalación

1. **Clonar y entrar al repositorio:**
   ```bash
   git clone <URL_DEL_REPOSITORIO>
   cd <NOMBRE_DEL_REPOSITORIO>
   ```

2. **Instalar dependencias:**
   ```bash
   npm install
   ```

## Ejecución

1. 
   ```bash
   npm run dev
   ```
---

Accede a http://localhost:5173 en tu navegador y presiona para comenzar a jugar.
